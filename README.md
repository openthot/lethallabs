# Lethal Labs Workspace

This repository contains the core website and documentation for **Lethal Labs**, an elite software firm developing premium SaaS solutions and maintaining open-source tools.

## Structure

- `index.html`: Homepage.
- `about.html`: Firm history and mission.
- `projects.html`: Portfolio of SaaS and open-source operations.
- `docs.html`: Operating models and insider information.
- `donate.html`: Supported cryptocurrency networks for contributions.
- `contact.html`: Enterprise inquiry and contact form.
- `styles.css`: Apple-inspired global design system.
- `wiki/`: Markdown contents for the repository's GitHub Wiki.

## Development

To view the website locally, you can serve the directory using any standard HTTP server:

```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000/index.html`.

## Deployment

This static site is designed to be hosted directly on GitHub Pages without any build steps or frameworks. Simply deploy the `main` branch to the root or `/docs` directory.
